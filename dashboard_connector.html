Troubleshooting

If you're having trouble connecting:

  1. Make sure the Streamlit server is running in your terminal
  2. Check for any error messages in the terminal
  3. Try restarting the server with: python launch_dashboard.py
  4. Ensure no firewall is blocking the connection
  5. Try a different browser

Manual Connection

You can also try entering one of these URLs directly in your browser address bar:

  • http://localhost:8501
  • http://127.0.0.1:8501
